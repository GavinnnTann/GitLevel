GitLevel

Level up your GitHub profile.

GitLevel turns your GitHub activity into an RPG-style character card — your language becomes your class, your contributions become XP and levels, and your followers and stars become fame. Drop it into your profile README and let your commit history speak in swords and spellbooks.

1Add it to your README

Paste this into your GitHub profile README and swap in your own username — that's the only thing you change:

![GitLevel](https://gitlevel.vercel.app/api/card?username=YOUR_GITHUB_USERNAME)

No token, no fork, no setup — you only supply your username. Cards are cached, so repeated README views stay fast.

2Pick a theme

Append &theme=NAME to the URL:

volt midnight sunset ice matrix transparent

3Tweak it

  • theme — color palette (see above)
  • hide_border — true / false
  • border_radius — corner rounding (0–60)
  • card_width — width in px (440–800)
  • animation — true / false
  • cache_seconds — cache TTL (3600–86400)
